# Post Comment REST API

## Tech Stack

-   Laravel 8
-   MySQL

## Installation

1. Clone this repository
2. cd `laravel-solid-pattern`
3. `composer install`
4. `cp .env.example .env`
5. `php artisan key:generate`
6. `php artisan migrate --seed`
7. `php artisan serve`
